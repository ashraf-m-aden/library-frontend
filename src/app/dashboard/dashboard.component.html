Hello, this is a little application I made to show off my Angular9 and springBoot skills.

It's a debut of a library application. In this little application, you can create new client, new types of books(genres) and then attribute some books to those types. Client can borrow books and you can register a book as "returned" if the client give it back.

For the front-end I used Angular, bootstrap, Angular material and a chart. This app had firebase as a backend before to execute CRUD operations, but i changed it to Spring boot.

The back-end now, use spring-rest, spring-data-jpa, hibernate, spring-security and jwt. But to visit, you have to authenticate first!
